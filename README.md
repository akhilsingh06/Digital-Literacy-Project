## 📋 Project Overview

This repository is my Digital Literacy Portfolio, created as part of the CSE0001 course assignment. As a **Student Digital Ambassador**, I have completed five tasks that cover the key pillars of digital literacy — from design and professional profiling to coding, communication, and cybersecurity awareness.

> **No prior coding knowledge was required** — every tool used is free and browser-based.
